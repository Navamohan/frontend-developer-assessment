NavMusicXboxUI

1) Initial Version UI html files give a basic search capability and display results to end user
2) Pages are populated with static search results to give end user a initial feel of results
3) User can hit on show/releases by clicking on releases 
4) User can shortlist their favourite artists in Last.Fm screen
5) User can add their favourite releases of the artist in MusicBrainz search page
6) All their favourites can be seen in seperate screen
7) Used angular js scope to keep track of the short list items, then stored in localstorage to show them favourites
